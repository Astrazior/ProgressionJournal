[h1]Progression Journal[/h1]

[b]A client-side progression helper for Terraria/tModLoader.[/b]

The mod adds an in-game journal with equipment recommendations for every supported class and progression stage. It provides a quick reference during gameplay without requiring you to constantly check a wiki.

[h2]Features[/h2]
[list]
[*]Equipment recommendations for every supported class at each progression stage
[*]Support for modded classes defined by profiles, such as Rogue
[*]Separate rating tiers: strong, usable, weak, and very weak
[*]Separate ratings for buffs
[*]Item acquisition details, including drops, shops, recipes, crafting stations, and required conditions
[*]Visual icons for events, moon phases, and other progression requirements
[*]A dedicated in-game journal interface
[*]Custom build creation, saving, importing, and exporting
[*]Build sharing through JSON or the in-game chat
[*]Support for items from other mods in custom builds
[*]Search and filtering tools for build setup
[*]Support for content mods through separate profiles
[/list]

[h2]Build System[/h2]

Starting with version [b]2.0.0[/b], the mod allows you to create custom builds for different classes and playstyles.

[img]https://i.imgur.com/PV7pm66.gif[/img]

Builds can include weapons, armor, accessories, consumables, and other equipment. They can be imported and exported through JSON or shared directly through the in-game chat.

Each build is linked to the active progression profile, class, and stage.

[h2]Profile System[/h2]

Starting with version [b]2.1.0[/b], the mod supports separate profiles for content mods. Profiles can be copied and edited directly in-game.

Instead of manually rating thousands of modded items, profiles use two main sections:

[list]
[*][b]Recommendations[/b] — official recommendations from the mod's sources, such as its wiki
[*][b]New Equipment and Buffs[/b] — items and buffs detected automatically through progression analysis
[/list]

[img]https://i.imgur.com/n7OK3RO.gif[/img]

The system analyzes bosses, loot, materials, recipes, and other progression connections. A simple progression chain looks like this:

[b]boss → new material → new armor or weapon[/b]

More complex chains may not always be detected correctly:

[b]boss → new enemy or new drop → material → equipment[/b]

As a result, automatically generated data may contain inaccuracies or become outdated after content mod updates.

[h2]Important[/h2]
[list]
[*]This is a [b]client-side[/b] mod, so the server does not need to install it
[*]The mod does not change progression or game balance and only works as a reference tool
[*]Wiki-based and automatically generated profiles may be incomplete or inaccurate
[/list]

[b]Supported mod profiles:[/b] Calamity, Thorium, Fargo's Souls, and Ancients Awakened: Classic.
