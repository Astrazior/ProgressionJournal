[h1]Progression Journal[/h1]

[b]Client-side progression helper for Terraria/tModLoader.[/b]

This mod adds an in-game journal with hand-picked equipment recommendations for each class at each progression stage.
It is meant to be a fast reference while playing, without opening a wiki every few minutes.

[h2]Features[/h2]
[list]
[*]Stage-by-stage equipment recommendations for melee, ranged, magic, and summoner
[*]Separate recommendation tiers: strong, usable, weak, and very weak
[*]Combat buffs page with stations, passives, potions, food, flasks, and permanent upgrades
[*]Item acquisition panel with drops, shops, recipes, crafting stations, and conditions
[*]Visual condition icons for events, moon phases, and other progression checks
[*]Fully usable in-game through a dedicated journal UI
[/list]

[h2]Important[/h2]
[list]
[*]This is a [b]client-side[/b] mod
[*]Servers do not need to install it
[*]It does not rebalance vanilla progression; it acts as a reference tool
[/list]
