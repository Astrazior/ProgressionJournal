[h1]Progression Journal[/h1]

[b]Client-side progression helper for Terraria/tModLoader.[/b]

[h2]EN[/h2]

This mod adds an in-game journal with hand-picked equipment recommendations for each class at each progression stage.
It is meant to be a fast reference while playing, without opening a wiki every few minutes.

[h2]Features[/h2]
[list]
[*]Stage-by-stage equipment recommendations for melee, ranged, magic, and summoner
[*]Separate recommendation tiers: strong, usable, weak, and very weak
[*]Combat buffs page with stations, passives, potions, food, flasks, and permanent upgrades
[*]Item acquisition panel with drops, shops, recipes, crafting stations, and conditions
[*]Visual condition icons for events, moon phases, and other progression checks
[*]Fully usable in-game through a dedicated journal UI
[/list]

[h2]Important[/h2]
[list]
[*]This is a [b]client-side[/b] mod
[*]Servers do not need to install it
[*]It does not rebalance vanilla progression; it acts as a reference tool
[/list]

[h2]RU[/h2]

Этот мод добавляет внутриигровой журнал с вручную подобранными рекомендациями по экипировке для каждого класса на каждом этапе прохождения.
Он задуман как быстрый справочник по ходу игры, чтобы не открывать вики каждые несколько минут.

[h2]Возможности[/h2]
[list]
[*]Рекомендации по экипировке для melee, ranged, magic и summoner на каждом этапе
[*]Отдельные уровни оценки: сильное, используемое, слабое и очень слабое
[*]Страница боевых бафов со станциями, пассивками, зельями, едой, флаконами и постоянными усилениями
[*]Панель получения предметов: дроп, магазины, рецепты, станции крафта и условия
[*]Визуальные иконки условий для событий, фаз луны и других проверок прогрессии
[*]Полностью используется прямо в игре через отдельный интерфейс журнала
[/list]

[h2]Важно[/h2]
[list]
[*]Это [b]клиентский[/b] мод
[*]Серверу его устанавливать не нужно
[*]Он не меняет баланс vanilla-прогрессии, а работает как справочный инструмент
[/list]
